# A Pytest Skeleton

Created by Rafael Dias, to help the start development in Python with Pytest (TDD).

## Pre-requirements:

* Docker
* Docker Compose

## Stack

* Python 3.6
* Pytest

## Start Container (bash)

```sh
$   sh bin/start_container.sh
```

## Run tests

```sh
$   sh bin/run_tests.sh
```
---
or 

```sh
$   sh bin/start_container.sh
```
and type
```sh
$   pytest
```
